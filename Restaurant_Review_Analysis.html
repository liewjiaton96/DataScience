Remove punctuations and stopwords in Review data¶

In [7]:
df = pd.read_csv("../DSproject/restaurant_review.csv")

review_filtered = []

for review in df['Review']:

  #Remove punctuations in raw 'Review'
  for character in review:
    if character in punctuations:
      review = review.replace(character, ' ')

  #Remove stopwords in raw 'Review'
  review = review.lower()
  tokens = review.split()  
  tokens_filtered = [t for t in tokens if t not in stop_words]
  review = ' '.join(tokens_filtered)
  review_filtered.append(review)
Remark:¶

1) Filtered 'Review' data is prepared for sentiment analysis where punctuations and stopwords are removed.

Remark:¶

Manual adjustment on the list of stopwords can be done to improve the relevancy. 2) Review sentiments have weak positive correlation to the rating from Tripadvisor users. This indicates that a single rating can not represent user's full expression and review sentiments should be taken into consideration when picking a restaurant.
